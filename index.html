Step-by-Step Guide

Follow these steps to get started with the dataset and Python scripts

1

Download & Extract

Download the dataset ZIP file and extract it to your project folder

Terminal 
unzip sentiment-analysis.zip
cd sentiment-analysis
2

Install Dependencies

Install required Python packages for full functionality

Terminal 
cd scripts
pip install -r requirements.txt

# Download NLTK data
python -c "import nltk; nltk.download('punkt'); nltk.download('stopwords'); nltk.download('wordnet')"
3

Generate Custom Data

Create your own dataset with desired size and parameters

Terminal 
# Generate 5000 balanced samples
python generate_data.py --samples 5000 --balanced

# Generate with train/test split (80/20)
python generate_data.py --samples 10000 --split 0.8 --all-formats
4

Preprocess Data

Clean and prepare text data for NLP models

Terminal 
# Basic preprocessing
python preprocess_data.py --input ../data/sentiment_data.csv

# Advanced: with lemmatization and stopword removal
python preprocess_data.py --input ../data/sentiment_data.csv \
    --lemmatize --remove-stopwords --build-vocab
5

Analyze Sentiment

Test sentiment analysis on your own text or evaluate datasets

Terminal 
# Interactive mode - type any text
python analyze_sentiment.py --interactive

# Evaluate accuracy on dataset
python analyze_sentiment.py --file ../data/test_data.csv --evaluate
6

Train ML Models

Train and compare multiple machine learning models

Terminal 
# Train all models and save the best one
python train_model.py --input ../data/sentiment_data.csv \
    --all-models --save --output ../models/

# Train specific model with cross-validation
python train_model.py --input ../data/sentiment_data.csv \
    --model svm --cross-validate 5
7

Visualize Results

Generate charts, word clouds, and comprehensive reports

Terminal 
# Generate all visualizations with HTML report
python visualize_data.py --input ../data/sentiment_data.csv \
    --all-charts --html-report --output ../charts/
